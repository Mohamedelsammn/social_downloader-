# downhub - Color Schema (Pure Black Edition)

## Core Palette
- **Primary Background:** `#000000` (Pure Black)
- **Primary Accent / CTA:** `#7C3AED` (Electric Purple)
- **Secondary Accent:** `#A78BFA` (Soft Violet Glow)
- **Surface Cards:** `#111111` (Elevated Black)
- **Borders / Dividers:** `#FFFFFF` at 10% opacity (Subtle Glass)

## Typography
- **Primary Text:** `#FFFFFF` (Pure White)
- **Secondary Text:** `#94A3B8` (Muted Lavender)
- **Muted / Disabled:** `#475569` (Slate)

## Functional Colors
- **Success:** `#10B981` (Emerald Green)
- **Error / Destructive:** `#EF4444` (Coral Red)
- **Downloading State:** `#7C3AED` (Progress Indicator)

## Platform Identities
- **Instagram:** Gradient `#C13584` to `#E1306C`
- **YouTube:** `#FF0000`
- **TikTok:** `#000000` with Cyan/Magenta accents
- **Facebook:** `#1877F2`
- **Twitter / X:** `#000000` with White icon
